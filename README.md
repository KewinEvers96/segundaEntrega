# segundaEntrega
segunda entrega de node


Para ejecutarlo, ubíquese en el directorio del proyecto con el CMD de Windows y escriba

	"node src/app"

Para visualizar la página, abra el navegador y escriba la dirección 

	"http://localhost:3000/"
